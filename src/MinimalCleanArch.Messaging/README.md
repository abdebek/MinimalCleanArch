# MinimalCleanArch.Messaging

Messaging and domain event helpers for MinimalCleanArch (Wolverine integration).

## Version
- 0.1.7 (net9.0). Use with `MinimalCleanArch` 0.1.7 and companions.

## What's included
- Domain event contracts and helpers.
- Wolverine integration extensions.
- DI extensions to wire messaging into your MinimalCleanArch app.

## Usage
```bash
dotnet add package MinimalCleanArch.Messaging --version 0.1.7
```

When using a local feed, add a `nuget.config` pointing to your local packages folder (e.g., `artifacts/nuget`) before restoring.
